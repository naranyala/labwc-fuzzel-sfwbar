# foot — Learning Material

> Source: `./sources/foot` | Upstream: https://codeberg.org/dnkl/foot

---

## What is foot?

**foot** is a fast, lightweight, Wayland-native terminal emulator. It has zero GTK/Qt dependency,
renders with pixman, and supports a server/daemon mode where a single process handles multiple
terminal windows with near-instant spawn time.

In OCWS, foot is the default terminal emulator launched from keybinds and the launcher.

---

## Features

- Wayland native (uses `xdg-shell` and `wl-seat`)
- Server daemon mode — one process, many windows
- Sixel image support
- Scrollback search
- Keyboard URL detection
- On-the-fly font resize and DPI adjustment
- IME support (`text-input-v3`)
- True color (24bpp) + styled underlines

---

## Usage

```bash
# Open a terminal window
foot

# Run a command in a terminal
foot -- bash -c "htop; exec bash"

# Server mode daemon (start once, open windows fast)
foot --server &

# Open a new window via the running server
footclient

# Pass a command to footclient
footclient -- nvim
```

## Configuration

`~/.config/foot/foot.ini` — OCWS provides `dotfiles/foot/foot.ini`.

```ini
[main]
font=JetBrains Mono:size=11
dpi-aware=yes
pad=12x8
shell=$SHELL

[scrollback]
lines=10000

[colors]
alpha=0.92
background=0f0f19
foreground=cdd6f4
# Catppuccin Mocha palette
regular0=45475a
regular1=f38ba8
regular2=a6e3a1
regular3=f9e2af
regular4=89b4fa
regular5=f5c2e7
regular6=94e2d5
regular7=bac2de
bright0=585b70
bright1=f38ba8
bright2=a6e3a1
bright3=f9e2af
bright4=89b4fa
bright5=f5c2e7
bright6=94e2d5
bright7=a6adc8

[cursor]
style=beam
blink=yes

[key-bindings]
search-start=Control+Shift+f
font-increase=Control+equal
font-decrease=Control+minus
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+C` | Copy |
| `Ctrl+Shift+V` | Paste |
| `Ctrl+Shift+F` | Start scrollback search |
| `Ctrl+=` | Increase font size |
| `Ctrl+-` | Decrease font size |
| `Ctrl+Shift+R` | Reload config |
| `Shift+PgUp/Dn` | Scroll |

## Build from Source

```bash
cd sources/foot
meson setup build --buildtype=release
ninja -C build
sudo ninja -C build install
```

**Dependencies:** pixman, wayland, xkbcommon, fcft, scdoc (optional: man pages)
